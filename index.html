Lab 8 – Node.js & Express

Overview

In Lab 8, I created a Node.js server using Express to serve both dynamic and static content. This lab introduced me to chaining fetch() promises, working with JSONPlaceholder API routes, and managing local servers.

Technologies Used

  • Node.js
  • Express
  • JavaScript Promises and fetch()
  • JSONPlaceholder API

What I Learned

I learned how to use Express middleware to serve static files, chain promises with .then() and .catch(), and build API routes that return dynamic JSON content.
